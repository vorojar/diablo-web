# 装备强化系统说明

## 1. 系统概述
位于 **罗格营地** 的铁匠 NPC **恰西** 提供装备强化服务。玩家可以通过消耗同类型的装备和金币来提升主装备的基础属性。

## 2. 强化规则
- **槽位布局**: 铁匠铺面板采用 "品" 字形布局。
    - **主槽位 (上)**: 放置需要强化的目标装备。
    - **祭品槽位 (下)**: 放置两个 **同部位** 且 **同稀有度** 的装备作为祭品。
- **消耗**:
    - **祭品**: 2件符合条件的装备 (无论强化成功或失败均会消耗)。
    - **金币**: 基础费用 1000G，随强化等级和装备稀有度递增。

## 3. 成功率与惩罚机制
强化并非百分百成功，高等级强化伴随风险：

| 强化等级区间 | 基础成功率 | 失败惩罚 |
| :--- | :--- | :--- |
| +0 → +1 | 100% | 无 |
| +1 → +5 | 逐步降低 (-10%/级) | 无 |
| +5 → +6 | 50% | 无 |
| +6 → +7 | 40% | **50% 概率等级 -1** |
| +7 → +8 | 30% | **50% 概率等级 -1** |
| +8 → +9 | 20% | **50% 概率等级 -1** |

> **注意**: 目前强化等级上限为 **+9**。

## 4. 属性提升
- 每次强化成功，装备的 **基础属性** (防御、最小伤害、最大伤害) 提升约 **10%**。
- 部分数值较大的附加属性也会有微小提升。
- 物品名称将获得 `+N` 后缀 (例如: `短剑 +3`)。

## 5. 操作指南
- **打开方式**: 在罗格营地与 NPC **恰西** 对话。
- **放入装备**:
    - **手动**: 打开面板后，点击背包中的装备即可自动填入对应槽位。
    - **自动**: 点击面板上的 "一键填充" 按钮，系统会自动从背包寻找合适的祭品填入。
- **开始强化**: 放入主装备和两个祭品，且金币足够时，点击 "开始强化" 按钮。
