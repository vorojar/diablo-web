# UI/UX 优化任务清单 (Diablo Web Remaster)

目标：消除“网页感”，打造沉浸式、厚重、史诗级的暗黑风格界面。

## 1. 全局视觉风格与纹理 (Visual & Atmosphere)
- [ ] **材质化背景**
  - 任务：将所有面板（角色、背包、技能等）的纯黑半透明背景替换为暗黑风格纹理。
  - 细节：使用深色石材、陈旧皮革或羊皮纸纹理，叠加噪点 (Noise) 增加质感。
- [ ] **装饰性边框**
  - 任务：替换 CSS 默认 `solid` 边框。
  - 细节：使用“金色/铜色哥特花纹”图片作为 `border-image`，或在面板四角添加独立的装饰性 SVG/图片元素。
- [x] **立体光影** ✅ v3.7
  - 任务：为所有窗口添加内阴影 (Inner Shadow) 和边缘高光。
  - 细节：模拟窗口厚度，使其看起来是浮在屏幕上的实体石板。

## 2. HUD (头部显示器) 重构
- [x] **拟真血球/蓝球**
  - 任务：优化 HP/MP 球体视觉效果。
  - 细节：
    - 添加高光层（玻璃反光）。
    - 添加内阴影层（球体体积感）。
    - (可选) 添加 CSS 动画模拟液体波动。
- [x] **一体化底座 (Action Bar)**
  - 任务：设计一个贯穿底部的金属/石质底座背景图。
  - 细节：将技能栏、腰带栏、经验条视觉上连接起来，不再是悬浮的方块。
- [ ] **哥特装饰填充**
  - 任务：在 HUD 空隙处添加暗黑元素（天使/恶魔雕像、锁链、符文）。

## 3. 交互反馈 (Juice & Feedback)
- [x] **濒危视觉警告 (Low HP Vignette)** ✅ v3.5
  - 任务：生命值 < 20% 时，屏幕边缘呼吸式闪烁红光。
  - 细节：使用 CSS `box-shadow: inset` 或覆盖层实现脉冲动画。
- [ ] **按钮交互质感**
  - 任务：优化技能和物品栏点击反馈。
  - 细节：
    - 点击时图标下沉/缩放效果。
    - 冷却 (CD) 效果改为扇形扫描遮罩，而非简单变灰。
- [ ] **自定义光标**
  - 任务：替换浏览器默认鼠标指针。
  - 细节：正常状态为金属手套，悬停可交互物体时变为金色/高亮状态。

## 4. 面板体验优化
- [ ] **统一关闭按钮**
  - 任务：重新设计面板右上角的 `X`。
  - 细节：使用红色宝石或金属风格的独立关闭按钮。
- [ ] **更清晰的装备状态**
  - 任务：优化不可用装备的视觉表现。
  - 细节：给予未辨识、属性不足的装备更明显的红色/灰色遮罩层。
- [x] **稀有度光效** ✅ v3.7
  - 任务：增强高品质装备的视觉辨识度。
  - 细节：暗金/套装物品在背包中显示微弱的背景呼吸光晕。

## 5. 字体与排版
- [ ] **信息层级优化**
  - 任务：区分标题与正文字体。
  - 细节：标题保持 `Cinzel`，密集数值文本尝试使用更易读的无衬线字体（如 Roboto/Lato）。
- [x] **关键词高亮** ✅ v3.7
  - 任务：在物品描述中通过颜色区分核心属性。
  - 细节：例如 <span style="color:#f00">火焰伤害</span>、<span style="color:#0f0">毒素抗性</span>。

---
**执行策略**：
建议按模块分批次执行，优先处理 **HUD** 和 **全局背景纹理**，这对视觉提升最明显。
